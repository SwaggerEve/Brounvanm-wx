Brounvanm 品牌用户成长平台
项目概述
Brounvanm 小程序不是一个传统的电商交易平台，而是一个以品牌服务和会员运营为核心的用户成长平台。通过 NFC 验真建立信任，企业微信建立连接，会员服务创造持续价值，最终形成完整的品牌运营闭环。

产品定位
品牌连接率 —— 衡量有多少购鞋用户完成了以下完整链路：

小程序目录结构
app.json	页面路由、TabBar、权限配置
utils/api.js	后端接口调用封装
utils/nfc.js	NFC 碰贴功能封装
pages/verify/	NFC 验真核心页面
pages/member/	品鉴官成长路径展示

二次访问小程序
产品本质
从"卖货工具"升级为"品牌用户成长平台"（Brand Membership Platform）

用户在这里经历的是：

购买产品 → 认识品牌 → 使用产品 → 参与品牌 → 认同品牌 → 推荐品牌
核心价值主张
传统电商小程序	Brounvanm 小程序
以成交金额为目标	以品牌连接率为目标
功能堆砌，首页复杂	聚焦验真+服务，路径清晰
会员=消费等级	会员=参与度+认同感
用户买完即走	用户成为品牌参与者

总结
Brounvanm 小程序的核心价值不在于"卖多少鞋"，而在于：

建立信任 —— NFC 验真让用户放心
建立关系 —— 企微连接让品牌有温度
创造持续价值 —— 会员服务让用户愿意回来
形成品牌共同体 —— 品鉴官让用户成为品牌一部分
当品牌连接和用户沉淀稳定后，商城和复购能力会是顺理成章的升级，而不是一开始就堆砌的功能。
